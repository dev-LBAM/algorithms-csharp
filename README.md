# Algorithms-CSharp

A collection of classic algorithms implemented in C# for study and practice. This project provides a simple console menu to run and test different algorithms interactively.

## Features

Currently implemented algorithms:

- Searching
  - Binary Search
- Sorting
  - Selection Sort

## How to Run

Clone the repository:

git clone https://github.com/dev-LBAM/algorithms-csharp.git
cd algorithms-csharp

Build and run the program:

dotnet run --project algorithmscsharp

## Console Menu

When you run the program, the main menu will appear:
```
== Algorithms ==
1 - Searching
2 - Sorting
0 - Exit
```

Selecting Searching displays:
```
== Searching Algorithms ==
1 - Binary Search
0 - Back to menu
```
Selecting Sorting displays:
```
== Sorting Algorithms ==
1 - Selection Sort
0 - Back to menu
```
Enter your choice to run the algorithm interactively.

## Purpose

This repository is for learning and experimentation. Each algorithm is implemented clearly to help understand how it works step by step. It's ideal for practicing:

- Problem-solving
- Algorithm logic
- C# syntax
